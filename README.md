### Сервис проверки доступа интернет-ресурсов

#### в качестве хранилища было принято решение сделать in-memory storage и cache.
#### Сервис, который принимает данные от пользователя и хранит их в памяти находится в cmd/api/data_service.
#### Сервис, проверяющий доступность интернет-ресурсов находится в cmd/api/task_service.

#### В случае недоступности/перезагрузки/остановки сервиса task_service, data_service кеширует данные, которые отправляются пользователем, но не обрабатываются.

#### При запуске сервиса task_service он проверяет кеш, если он не пуст, то проверяет ссылки на доступность и отправляет в data_service.

#### data_service удаляет из кеша обработанные данные и сохраняет в storage.
